# How to import backup

### You should save the backup file into Files App first.

#### 1. Open Dagon and Tap button on the left top.
![Guide101](/images/Guide101.jpg)

#### 2. Tap "Settings" button on the right top.
![Guide102](/images/Guide102.jpg)

#### 3. Tap "Import From Files" Row. Files App will pop up, then you should select the backup file and Dagon will import the backup data.
![Guide103](/images/Guide103.jpg)
