# How to export backup

#### 1. Open Dagon and Tap button on the left top.
![Guide101](/images/Guide101.jpg)

#### 2. Tap "Settings" button on the right top.
![Guide102](/images/Guide102.jpg)

#### 3. Tap "Backup Files" Row.
![Guide103](/images/Guide103.jpg)

#### 4. Tap the backup row you want to export.
![Guide104](/images/Guide104.jpg)

#### 5. Save the backup file to anywhere you want to.
![Guide105](/images/Guide105.jpg)
