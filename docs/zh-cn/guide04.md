# 如何保存Instagram的图片

#### 1. 打开Instagram客户端，点选您要保存的帖子的右上角按钮
![Guide201](/../images/Guide201.jpg)

#### 2. 选择“Share To...”
![Guide202](/../images/Guide202.jpg)

#### 3. 点选“加入Dagon”项
![Guide203](/../images/Guide203.jpg)

#### 4. 点选“保存”按钮，记得打开“保存所有图片”
![Guide204](/../images/Guide204.jpg)

#### 5. 打开Dagon，Instagram的帖子会出现在列表中
![Guide205](/../images/Guide205.jpg)

#### 6. 您可以点选导航栏中新建按钮旁边的图库按钮，打开图库然后浏览您刚刚保存的图片
![Guide206](/../images/Guide206.jpg)
