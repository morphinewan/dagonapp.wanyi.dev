# 如何从Safari共享网址

#### 1. 打开Safari，点击动作按钮
![Guide001](/../images/Guide001.jpg)

#### 2. 您会看到iOS的动作表单
![Guide002](/../images/Guide002.jpg)

#### 3. 您会看到“加入Dagon”菜单，随后点击"编辑操作"按钮
![Guide003](/../images/Guide003.jpg)

#### 4. 您可以重新排列项目的顺序，把Dagon的菜单条移动到最顶部
![Guide004](/../images/Guide004.jpg)

#### 5. 点击“加入Dagon”菜单项
![Guide005](/../images/Guide005.jpg)

#### 6. 点击“保存”按钮
![Guide006](/../images/Guide006.jpg)
